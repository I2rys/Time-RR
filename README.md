## Installation
Github:

    git clone https://github.com/I2rys/Time-RR
    
NPM Packages:
```
npm i discord.js
```

## Setup
1. Open index.js
2. Find **Self** > **access** put your Discord account ID there.
3. Find **Self** > **discord_token** put your Discord bot token there.

## Usage

    node index.js
